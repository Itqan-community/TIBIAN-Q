# Frontend Developer Volunteer

## Role Overview
The Frontend Developer Volunteer builds beautiful, responsive, and performant user interfaces for ITQAN's Quranic technology products. This role implements designs in React and TypeScript, ensuring excellent user experiences across web and mobile platforms while maintaining code quality and accessibility standards.

## Key Responsibilities

### UI Development (50% of time)
- Build responsive web interfaces using React, TypeScript, and Next.js
- Implement designs from Figma with pixel-perfect accuracy
- Create reusable UI components and component libraries
- Develop RTL (right-to-left) interfaces for Arabic content
- Ensure cross-browser compatibility and responsiveness
- Optimize performance and bundle sizes

### Feature Implementation (25% of time)
- Collaborate with Product Managers on feature requirements
- Work with Backend Developers on API integrations
- Implement state management (Redux, Zustand, React Query)
- Build forms, modals, and complex UI interactions
- Integrate third-party libraries and services
- Write clean, maintainable code

### Code Quality & Testing (15% of time)
- Write unit tests and integration tests (Jest, React Testing Library)
- Conduct code reviews for other developers
- Follow coding standards and best practices
- Optimize component performance and rendering
- Ensure accessibility (WCAG standards)
- Document code and components

### Collaboration (10% of time)
- Participate in sprint planning and daily standups
- Collaborate with designers on implementation feasibility
- Communicate with backend team on API contracts
- Support QA team with bug fixes and testing
- Share knowledge with team members
- Provide technical feedback on designs

## Required Skills

### Core Technologies
- **React**: Advanced proficiency with React 18+, hooks, context
- **TypeScript**: Strong typing skills and type safety
- **Next.js**: SSR, SSG, routing, and API routes
- **CSS/Styling**: TailwindCSS, CSS-in-JS, or styled-components
- **HTML**: Semantic HTML5 and accessibility

### Frontend Tools
- **State Management**: Redux, Zustand, or React Query
- **Build Tools**: Vite, Webpack, or similar
- **Testing**: Jest, React Testing Library, Playwright
- **Version Control**: Git, GitHub workflows
- **Package Managers**: npm, yarn, or pnpm

### UI/UX Implementation
- **Responsive Design**: Mobile-first, adaptive layouts
- **RTL Support**: Right-to-left interface implementation
- **Accessibility**: ARIA, keyboard navigation, screen readers
- **Performance**: Code splitting, lazy loading, optimization
- **Animations**: Framer Motion, CSS animations

### Domain Knowledge
- **Arabic Text**: Handling Arabic fonts and RTL layout
- **Quranic Content**: Respectful presentation of sacred text
- **Islamic UI Patterns**: Prayer times, Qibla direction, etc.
- **Muslim Users**: Understanding of user preferences

### Soft Skills
- Strong problem-solving abilities
- Attention to detail and quality
- Collaborative team player
- Self-motivated and proactive
- Good communication skills
- Ability to work async
